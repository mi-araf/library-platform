# 📚 Araf's Library

Araf's Library is a modern online library platform where users can discover, browse, and explore books in a clean and user-friendly digital experience.

The project is designed to help users find books easily, view important book information, and enjoy a simple library browsing system with authentication and responsive design.

## 📝 Project Description

Araf's Library is an online book browsing and library management platform built with Next.js and MongoDB. It allows users to explore books, view book details, search and filter books, and access user-related features through authentication.

The goal of this project is to create a smooth, responsive, and practical library experience for users who want to discover and manage books online.

---

## ✨ Key Features

### 📖 Browse Books
Users can explore a collection of books with useful information such as book title, author, category, and available quantity.

### 🔍 Search and Filter
Users can search for books and filter them by category to find their desired books quickly.

### 🔐 Authentication System
The platform includes user authentication features, allowing users to register, log in, and access protected pages.

### 👤 User Profile
Logged-in users can access their profile and view personal information related to their account.

### 📙 Access book
To access book and see its details, user needs to sign-in first.

## ⚙️ Installation and Setup

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/mi-araf/library-platform.git
```

### 2. Go to the Project Folder

```bash
cd library-platform
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Create Environment Variables

Create a `.env.local` file in the root directory and add your environment variables.

```env
MONGODB_URI=your_mongodb_connection_string
BETTER_AUTH_SECRET=your_better_auth_secret
BETTER_AUTH_URL=http://localhost:3000

GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```

### 5. Run the Development Server

```bash
npm run dev
```

### 6. Open in Browser

Visit:

```bash
http://localhost:3000
```
